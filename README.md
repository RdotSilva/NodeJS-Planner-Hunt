# NodeJS Planner Hunt

Find new planner roles and send alerts when they are found

## Installation

1. Install dependencies in main project folder.

```
npm install
```

2. Create ENV file with appropriate values

## Built With

- JavaScript
- NodeJS
- [Twilio](https://www.twilio.com/)
- [NeDB](https://dbdb.io/db/nedb)
- VSCode
